## Overview

`mx8.mix()` blends multiple loaders deterministically under one shared memory envelope. Use it to train on weighted combinations of datasets without managing interleaving yourself.

## Basic Usage

```python
import mx8

loader_a = mx8.load("s3://bucket/a@refresh", batch_size_samples=32, max_ram_gb=6)
loader_b = mx8.load("s3://bucket/b@refresh", batch_size_samples=32, max_ram_gb=6)

mixed = mx8.mix(
    [loader_a, loader_b],
    weights=[0.7, 0.3],
    seed=17,
    epoch=3,
    source_exhausted="error",
    max_ram_gb=12,
    profile="balanced",
)

for batch in mixed:
    train_step(batch)
```

## Parameters

### Source Loaders

<ParamField path="loaders" type="list" required>
  List of source loaders. Accepts loaders created by `mx8.load(...)` and `mx8.DistributedDataLoader(...)`.
</ParamField>

<ParamField path="weights" type="list[float]" required>
  List of positive floats for weighted sampling. Values are normalized internally.
</ParamField>

### Determinism

<ParamField path="seed" type="int" required>
  Random seed for deterministic scheduling
</ParamField>

<ParamField path="epoch" type="int" required>
  Epoch number for deterministic interleaving (used with `seed`)
</ParamField>

### Exhaustion Policy

<ParamField path="source_exhausted" type="str" default="error">
  Behavior when a source runs out:
  - `"error"`: fail fast (default)
  - `"allow"`: drain remaining sources
</ParamField>

<ParamField path="starvation_window" type="int">
  Starvation accounting window for scheduler
</ParamField>

### Memory Management

<ParamField path="max_ram_gb" type="int | float" required>
  Memory budget for the mixed loader. All sources share one inflight cap — backpressure is global.
</ParamField>

<ParamField path="profile" type="str" default="balanced">
  Safety/throughput preset: `"safe"`, `"balanced"`, or `"throughput"`
</ParamField>

### Checkpoint & Resume

<ParamField path="resume_from" type="str">
  Checkpoint token from `mixed.checkpoint()`. Resume contract:
  - `seed`, `epoch`, and source count must match
  - Restores scheduler position and per-source delivery counters
  - If source checkpoints differ, continues in best-effort mode and increments `mix_resume_source_checkpoint_mismatch_total`
</ParamField>

<ParamField path="job_id" type="str">
  Job identifier (accepted for API consistency, currently ignored)
</ParamField>

<ParamField path="cluster_url" type="str">
  Coordinator URL (accepted for API consistency, currently ignored)
</ParamField>

    ```python
            weights=[0.7, 0.3],
            seed=seed,
            epoch=epoch,  # Different mix each epoch
            max_ram_gb=12,
        )
        
        for batch in mixed:
            train_step(batch)
    ```

## Deterministic Ordering

For a fixed set of:

- Manifest hashes
- Weights
- Seed
- Epoch
- Frozen membership

The mixed stream order is **deterministic and replayable**.

```python
# Same mix order every time with same inputs
mixed = mx8.mix(
    [loader_a, loader_b],
    weights=[0.7, 0.3],
    seed=42,
    epoch=0,
    max_ram_gb=12,
)
```

Change `epoch` to get a different deterministic order:

```python
# Different order, still deterministic
mixed = mx8.mix(
    [loader_a, loader_b],
    weights=[0.7, 0.3],
    seed=42,
    epoch=1,  # Different epoch → different order
    max_ram_gb=12,
)
```

```python
stats = mixed.stats()

# Per-source diagnostics
for src in stats["mix_sources"]:
    print(f"Source manifest: {src['manifest_id']}")
    print(f"Delivered samples: {src['delivered_samples_total']}")
    print(f"Delivered batches: {src['delivered_batches_total']}")
    print(f"Weight: {src['weight']}")
```

## Checkpoint & Resume

```python
import mx8

loader_a = mx8.load("s3://bucket/a/", batch_size_samples=32, max_ram_gb=6)
loader_b = mx8.load("s3://bucket/b/", batch_size_samples=32, max_ram_gb=6)

mixed = mx8.mix(
    [loader_a, loader_b],
    weights=[0.7, 0.3],
    seed=42,
    epoch=0,
    max_ram_gb=12,
)

for step, batch in enumerate(mixed):
    train_step(batch)
    
    if step % 1000 == 0:
        token = mixed.checkpoint()
        save_checkpoint(model, token)

# Resume from checkpoint
loader_a = mx8.load("s3://bucket/a/", batch_size_samples=32, max_ram_gb=6)
loader_b = mx8.load("s3://bucket/b/", batch_size_samples=32, max_ram_gb=6)

mixed = mx8.mix(
    [loader_a, loader_b],
    weights=[0.7, 0.3],
    seed=42,
    epoch=0,
    resume_from=token,
    max_ram_gb=12,
)
```

## Shared Memory Envelope

All source loaders share one inflight cap:

```python
# Each source gets 6GB, but mix enforces global 12GB cap
loader_a = mx8.load("s3://bucket/a/", batch_size_samples=32, max_ram_gb=6)
loader_b = mx8.load("s3://bucket/b/", batch_size_samples=32, max_ram_gb=6)

mixed = mx8.mix(
    [loader_a, loader_b],
    weights=[0.7, 0.3],
    seed=42,
    epoch=0,
    max_ram_gb=12,  # Global cap, not 6+6
)
```

Backpressure is global across all sources.

## Best Practices

<Check>
**Pin snapshots for multi-epoch reproducibility** — ensures same data across epochs
</Check>

<Check>
**Use `source_exhausted="error"` (default) for balanced datasets** — catches unexpected exhaustion early
</Check>

<Check>
**Set `source_exhausted="allow"` for imbalanced datasets** — drains larger datasets after smaller ones finish
</Check>

<Check>
**Monitor `mix_sources` stats** — track per-source delivery and balance
</Check>

<Check>
**Change `epoch` parameter across training epochs** — ensures different mix order each epoch
</Check>

<Warning>
**All sources share one memory budget** — set `max_ram_gb` for the mix, not per-source totals
</Warning>

<Warning>
**Resume requires matching `seed`, `epoch`, and source count** — strict validation enforced
</Warning>
